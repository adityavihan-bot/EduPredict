# EduPredict
EduPredict is an intelligent Machine Learning powered web application designed to predict a student’s end examination performance based on academic behavior and daily study patterns.

This project leverages the power of Linear Regression from Scikit-learn to analyze multiple student-related parameters such as:

📚 Study Hours Per Day
🏫 Attendance Percentage
😴 Sleep Hours Per Night
📝 Previous Academic Score

Using these inputs, the system predicts the expected examination marks and provides instant performance analysis through an interactive and user-friendly Streamlit interface.
